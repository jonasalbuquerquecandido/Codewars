# 🧠 Codewars Practice

Repositório dedicado à prática contínua de lógica de programação e resolução de problemas através de desafios da plataforma Codewars.

---

## 🚀 Objetivo

Desenvolver e fortalecer habilidades essenciais para programação:

* Raciocínio lógico
* Resolução de problemas
* Pensamento algorítmico
* Escrita de código limpo
* Preparação para desafios técnicos

---

## 🛠️ Tecnologias utilizadas

* JavaScript
* C++

---

## 📁 Estrutura do repositório

Os exercícios estão organizados por nível de dificuldade:

```
codewars/
├── 8kyu/   # Nível iniciante
├── 7kyu/   # Nível básico/intermediário
├── 6kyu/   # Nível intermediário
```

---

## 🧩 Estrutura dos exercícios

Cada exercício contém:

* 🧩 Nome do problema
* 🔗 Link para o desafio original
* 💡 Estratégia utilizada
* 💻 Implementação da solução

### 📌 Exemplo:

```javascript
// 🧩 Problema: Even or Odd
// 🔗 https://www.codewars.com/...

// 💡 Estratégia:
// Verificar se o número é divisível por 2

function evenOrOdd(number) {
  return number % 2 === 0 ? "Even" : "Odd";
}
```
